# BrightLearn Data Analytics – Research Assignment 1

## Overview
This repository contains my completed research assignment for the BrightLearn Data Analytics course. The assignment covers foundational concepts in data analytics, artificial intelligence, APIs, programming tools, cloud computing, and data ethics.
## Topics Covered
### Data Fundamentals
Covers the basics of data systems including databases, data warehouses, data lakes, data marts, types of data, data modeling, data mining, DBMS platforms, ETL processes, and data storage formats
### Artificial Intelligence & Machine Learning
Explores AI concepts, machine learning types, deep learning, neural networks, NLP, large language models, generative AI, training data, bias in AI, and computer vision.
### APIs & Developer Tools
Includes APIs, API keys, REST APIs, JSON format, prompt engineering, programming concepts (map function), cloud computing models, Git version control, Jupyter notebooks, and Python in data analytics.
### Data Ethics & Careers
Focuses on data privacy (GDPR), data ethics, data governance, AI bias, BI tools (Power BI, Tableau), and career roles in data (Data Analyst, Data Scientist, Data Engineer, AI Engineer)  
### Reflection
Personal reflection on key learnings, challenges, and the future impact of data and AI on careers and everyday life
## Key Learnings
- Understanding how data is stored, processed, and analyzed  
- Fundamentals of AI, ML, and data systems  
- Importance of APIs, cloud computing, and programming tools  
- Ethical use of data and AI in real-world applications  
- Career pathways in data and artificial intelligence
